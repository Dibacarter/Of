Page Structure and Flow:
1. Page Layout: The page is designed to center the content vertically and horizontally, creating a clean and focused form for users to input their information.
2. Form Structure: The form collects the user’s name and email. When the user submits the form, it sends the data to a specified endpoint (Formspree in this case).
3. CSS Styling: The CSS styles ensure the form elements are visually consistent and easy to use. It also styles the text and the overall layout to match your branding.

How the Page Works:
1. HTML Content:

  • The HTML contains the structure of the page, including the title, meta tags, and form elements.

  • The form includes input fields for the user’s email and name, and a submit button.

    • 2. CSS Styling:
  • The CSS styles the body, form, labels, input fields, and buttons to ensure a cohesive look.

  • It uses flexbox for centering and layout management.

  • Additional styles are added to ensure padding and margins are consistent across elements.

    • 3. Form Submission:
  • When a user fills out the form and clicks the “SEND” button, the form data is sent to the specified endpoint using the POST method.

  • In this case, the form action is set to a Formspree endpoint (https://formspree.io/f/mldrezaz), which will handle the form submission and send the data to your email.
